##  Twodo-li 서비스 소개
챗바퀴같은 일상속에서 **Twodo-li가 즐거움을 줄 수 있다면**,<br/>
Twodo-li가 지루한 Todolist달성을 도와드릴게요.<br/>

Twodo-li서비스는 `Todo를 친구와 작성하고 서로가 목표달성을 위한 동료가 됩니다.` <br/>
그 과정에서 자신의 `캐릭터도 자유롭게 꾸밀 수 있어요.`
